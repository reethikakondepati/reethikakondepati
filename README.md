## 🚀 Featured Project

### 🏥 MediScan AI — Medical Report Simplifier

🔗 **Live Demo:**  
👉 https://reethika1519-ai-medireport.hf.space/

📌 **Project Description:**  
An AI-powered web application that simplifies complex medical reports into easy-to-understand insights using OCR and NLP techniques.

✨ **Key Features:**
- 📄 Multi-format file upload (PDF, Image, TXT)  
- 🔍 OCR-based text extraction using Tesseract  
- 🤖 AI-driven medical report analysis  
- 📊 Severity classification (Mild / Moderate / Critical)  
- 🚨 Smart alert system for critical cases  
- 📈 Comparative analysis of previous reports  
- 📂 User dashboard with report history  

🛠️ **Tech Stack:**  
Python, Flask, HTML, CSS, JavaScript, GPT API, Tesseract OCR, SQLite
